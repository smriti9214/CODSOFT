# 🧑‍💻 CodSoft Internship Python Projects

This repository contains three beginner-friendly Python projects developed during my **CodSoft Internship**. These projects helped me strengthen my understanding of Python fundamentals, logic building, and problem-solving skills.

---

## 📌 Projects Included

### 1. 🔐 Random Password Generator

A simple Python program that generates strong and secure passwords based on user preferences.

**Features:**

* Generates random passwords
* Customizable length
* Includes letters, numbers, and special characters
* Helps improve security awareness

**Tech Used:**

* Python
* `random` module
* `string` module

---

### 2. 🧮 Calculator

A basic command-line calculator that performs arithmetic operations.

**Features:**

* Addition, Subtraction, Multiplication, Division
* User-friendly input system
* Continuous operations until exit

**Tech Used:**

* Python
* Conditional statements
* Functions

---

### 3. ✊✋✌️ Rock Paper Scissors Game

A fun command-line game where the user plays against the computer.

**Features:**

* User vs Computer gameplay
* Random choice generation
* Score tracking
* Replay option

**Tech Used:**

* Python
* `random` module
* Loops and conditionals

## 🚀 How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/codsoft-python-projects.git
   ```

2. Navigate to the project folder:

   ```bash
   cd codsoft-python-projects
   ```

3. Run any project:

   ```bash
   python filename.py
   ```
